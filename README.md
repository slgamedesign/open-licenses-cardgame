## 📖 About the Project

**Open Licenses** is an analog, tabletop card game developed to teach the fundamentals of intellectual property in the digital age. Instead of writing code, players sit around a table and compete to build the best portfolio of legally sound projects.

This project was developed as an assignment for the **Technical English III** (*Inglês Técnico III*) course. It offers a tactile, screen-free alternative to learning about software and media licensing.

---

## 🎮 Gameplay & Objective

The main objective is to be the first player to form two complete sets of cards representing different types of Open Source and Creative Commons licenses. 

Players take turns matching cards in the discard pile and building specific combinations on their table:
* **Pairs:** e.g., CC + GPL.
* **Trios:** e.g., GPL + Public Domain + CC.
* **Four of a Kind:** e.g., CC + BY + NC + ND.

### ⚡ Action Cards
To win, you must also use strategies and deal with special events. The game includes 3 types of Action Cards that can be played instantly:
* **H4CK:** Disrupt your opponents' sets.
* **BUG FIX:** Protect your licenses or recover from a hack.
* **EXPANSION:** Broaden your set possibilities.

## 🖨️ How to Play (Print & Play)

Since this is an analog game, you don't need to install anything. Just print, cut, and play!

1. Go to the files in this repository.
2. Download the [**CARTAS_FINAL.pdf**](./CARTAS_FINAL.pdf) file (which contains the beautifully designed card deck).
3. Print the document on high-quality, thick paper (like cardstock) for a better shuffling experience.
4. Cut out the individual cards.
5. Download the [**CARD GAME OPEN LICENSES.pdf**](./CARD%20GAME%20OPEN%20LICENSES.pdf) file to read the full rulebook.
6. Shuffle the deck, gather your friends, and start playing!
